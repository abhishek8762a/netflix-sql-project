# Netflix SQL Analysis Project

## 📌 Objective
This project analyzes a Netflix dataset using SQL to extract meaningful insights about content, ratings, and trends.

## 🛠️ Tools Used
- MySQL Workbench  
- SQL  

## 📂 Dataset
- `netflix.csv` (contains Netflix shows/movies data)

## 🔍 Key Analysis Performed
- Rating distribution analysis  
- Top-rated shows identification  
- Year-wise content analysis  
- Average rating calculation  
- High-quality content filtering (>90 rating)  
- Score categorization (Excellent, Good, Average, Low)  
- Content distribution across rating categories  

## 📊 Key Insights
- TV-14 category has the highest number of shows  
- Average rating is around 84, indicating strong overall quality  
- 2016 had the highest number of releases  
- 2017 had the highest average rating (~90)  
- Majority content falls under 'Excellent' category  
- High-quality content (>90 rating) forms a significant portion  

## 💡 Recommendations
- Increasing high-quality content (rating > 90) can improve user satisfaction  
- Focus on TV-14 category as it has both high volume and quality  
- Maintain consistent content release strategy as seen in past trends  

## 📁 Files Included
- `netflix_analysis.sql` - SQL queries with insights  
- `netflix.csv` - Dataset used for analysis
